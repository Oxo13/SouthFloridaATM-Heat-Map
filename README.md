# SouthFloridaATM-Heat-Map
# created a large dataset through google maps api and US Census data to address ATM disparities among different regions and economic standings within Palm Beach, Broward and Miami-Dade counties. 
file was entirely created using Google Colab and Gemini and Claude through "vibe coding". Initiative to show that anyone can code with AI and create unique datasets through real-world observation within their community.
